# 记忆训练游戏

# A simple game to train your memory

本项目用Arduino搭建的一个简单的记忆训练游戏机，可以在你闲暇的时候帮你训练短时记忆能力。

This is a memory train game built by Arduino. It can train your memory in your free time.
